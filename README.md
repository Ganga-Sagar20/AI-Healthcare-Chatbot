# AI Healthcare Chatbot - Major Project

A modern, AI-powered healthcare chatbot built using Python, Streamlit, and NLTK for symptom detection and basic medical recommendations.

## Features
- **Modern UI**: Built with Streamlit featuring a clean, responsive "blue healthcare" theme.
- **NLP Preprocessing**: Utilizes NLTK for text cleaning, tokenization, and stopword removal.
- **Symptom Checker**: Matches user descriptions with a built-in knowledge base (`symptom_data.csv`).
- **Emergency Detection**: Identifies critical keywords to immediately advise seeking emergency help.

## Tech Stack
- Frontend: Streamlit
- Backend: Python
- NLP: NLTK
- Data: Pandas (CSV)

## How to Run Locally

1. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Run the Streamlit application:**
   ```bash
   streamlit run app.py
   ```
